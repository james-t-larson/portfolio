Blog

This is a template for a blog. This template creates blog posts based on the number of markdown files in a specific directory. I utilized GatsbyJS, NodeJS, and SCSS for this project. The site pulls the filename to create a slug, and adds the title, media and text content to the webpage. It requires more work as far as styling, but at the moment I enjoy its simplicity.
